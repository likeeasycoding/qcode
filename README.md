# QuickCode开发软件

#### 软件介绍：
低代码开发平台(Qcode),设计的初衷主要是为了解决项目开发中繁琐的代码复制粘贴，繁琐的增删改查的基础操作。
系统集成了springboot和vue-element-admin,通过程序可以在本地启动一个服务，并且一键自动生成项目开发框架代码,根据本地数据库结构，自动生成springboot和vue代码。
